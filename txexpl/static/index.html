Drag the button to your bookmarks bar, click the bookmark on etherscan transaction URL to decode and explain the transaction.

Note: This is a PoC demo, which is a bit slow.

Transaction Explainer
Examples:

explain_url(url): Decode the transaction from etherscan tx URL or safe multi-sig tx URL.

explain_txid(chain, txid): Decode the transaction by chain and txid

explain_call(chain, to, data, value): Decode with raw to, calldata and value

Temper-monkey script to support decoding on signing with browser wallet.
